隐私政策 (Privacy Policy)

生效日期： 2026年2月10日

欢迎使用 AiToWords（以下简称“本工具”）。我们非常重视您的隐私。本隐私政策旨在向您说明我们如何处理您的数据。

1. 数据处理方式

核心承诺：AiToWords 是一个纯客户端工具。

  • 本地处理： 所有的 Markdown 解析、渲染及 Word 文档生成过程均在您的浏览器（客户端）本地完成。
  • 无数据上传： 您输入的任何文本、代码或图片链接及其转换后的内容，不会被上传到我们的服务器或任何第三方服务器。
  • 无数据库： 我们不维护任何数据库来存储您的对话内容。

2. 信息收集

由于本工具的离线优先特性，我们不收集任何个人身份信息（PII），例如您的姓名、邮箱或电话号码。

第三方服务

为了维持本工具的免费运营及优化用户体验，我们使用了以下第三方服务，它们可能会收集非身份识别的使用数据（如浏览器类型、访问时长）：

  • Google AdSense： 用于展示广告。Google 可能会使用 Cookie 根据您之前的访问记录展示个性化广告。您可以访问 Google 的广告设置页面来管理您的偏好。
  • GitHub Pages： 本网站托管于 GitHub Pages。GitHub 可能会收集必要的服务器日志以确保服务安全和稳定。

3. Cookie 的使用

本工具本身不使用 Cookie 来跟踪您的行为。第三方服务（如 Google AdSense）可能会使用 Cookie 来提供广告服务。您可以通过浏览器设置随时清除或禁用 Cookie。

4. 外部链接

本工具可能包含指向第三方网站（如 GitHub 仓库）的链接。我们对这些第三方网站的隐私惯例不承担责任。建议您在离开本站后阅读相关网站的隐私政策。

5. 政策更新

我们可能会不时更新本隐私政策。任何更改都将在此页面上发布，且生效日期将相应更新。继续使用本工具即表示您接受更新后的隐私政策。

6. 联系我们

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 联系我们：https://github.com/Huangyiyi-Li/ai2word/issues
