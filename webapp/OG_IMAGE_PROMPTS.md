## 📝 使用说明

1. **复制每个 Prompt** 到 Nano Banana
2. **生成图片后**，保存为对应的文件名（如 `og-image.png`）
3. **放置路径:** `webapp/assets/` 目录
4. **完成后**，推送到 GitHub：

```bash
cd /Users/szjxxiangmubu/.openclaw/workspace/aitowords-repo
mkdir -p webapp/assets
# 将生成的图片放到 webapp/assets/ 目录
git add webapp/assets/
git commit -m "SEO: 添加OG图片"
git push origin main
```
