# 安装页面需求文档 (PRD)

## 1. 背景

用户需要安装浏览器插件，但目前插件尚未上架商店。需要提供一个离线安装包的下载和安装引导页面。

## 2. 需求描述

1. **独立安装页**：在 `webapp` 下创建一个新的 `install.html` 页面。
2. **下载功能**：提供 ZIP 格式的插件压缩包下载。
3. **安装教程**：
    * 针对 Chrome 和 Edge 浏览器提供不同的文字引导。
    * 指导用户开启「开发者模式」。
    * 指导用户「加载已解压的扩展程序」。
4. **入口优化**：将首页 (`index.html`) 的「安装插件」按钮链接指向该页面。

## 3. 功能列表

| 功能模块 | 详细说明 | 优先级 |
| :--- | :--- | :--- |
| 下载插件 | 点击按钮直接下载 `AiToWords_Extension.zip` | P0 |
| Chrome 教程 | 包含地址栏输入 `chrome://extensions`，开启开发者模式，拖拽或加载安装 | P0 |
| Edge 教程 | 包含地址栏输入 `edge://extensions`，开启开发者模式，加载解压 | P0 |
| 首页入口 | 首页按钮点击跳转至此页 | P0 |

## 4. 界面设计

- 保持与首页一致的 UI 风格（配色、字体）。
* 清晰的步骤指引 (Step 1, Step 2...)。
